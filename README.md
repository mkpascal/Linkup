Linkup
======

Linkup is a two-way message system using node.js & WebSocket.

## Usage ##

### Server ###

You need to install the ws module (node.js),

`npm install ws`

Then,

`node server.js`

### Client ###

Run the page,

`client.html`

Don't forget to modify the Websocket server before using it.
